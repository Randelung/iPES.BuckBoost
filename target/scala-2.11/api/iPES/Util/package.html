    Coordination of animations for circuit diagrams.

    Circuit elements may register themselves here, and the animator will trigger each animation callback once a tick. Elements may also unregister themselves.

    When instantiating the class, two canvas contexts need to be supplied, one for the mostly unchanging circuitry, and one where the actual animation takes place.
